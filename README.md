# Student Wellbeing Analysis

This project analyzes student wellbeing factors such as depression, academic pressure, work pressure, and study satisfaction.  
The dataset is cleaned and visualized using **NumPy**, **Pandas**, and **Matplotlib**.  

## Current Features
- Age grouping and statistical insights  
- Depression rate by age group  
- Work Pressure and Academic Pressure comparison  
- Multiple bar charts for analysis
